# pomodoro

## Project setup

```bash
yarn install
```

### Run Vite dev server

```bash
yarn dev
```

### Develop the Tauri desktop app

```bash
yarn tauri:dev
```

### Build for production (web)

```bash
yarn build
```

### Build the Tauri desktop app

```bash
yarn tauri:build
```

### Lint source files

```bash
yarn lint
```
